## Overview

AutoRide is a single-page car rental web application that lets users browse vehicle options, compare rental plans, and submit a booking request through a guided, form-based reservation flow. The product is designed around a confident, modern visual identity: a dark interface paired with a high-contrast lime green accent that signals speed and energy without sacrificing legibility or usability.

The application is organized as a set of independent, composable React components, each paired with its own scoped stylesheet. A shared set of CSS custom properties (color, spacing, and typography tokens) keeps the visual language consistent from the navigation bar down to the footer, making the codebase straightforward to theme, extend, or hand off to another developer.

## Features

| Section | Description |
|---|---|
| Navigation bar | Sticky, responsive navbar with active link state and accessible focus styling |
| Hero | Landing section with headline, value proposition copy, and an inline booking form |
| Booking form | Car type, pickup and drop-off location, and rental/return date selection |
| Feature highlights | Numbered breakdown of core service benefits |
| Media and copy split | Image-led section pairing visuals with a numbered feature list |
| Stats | Animated metric cards with custom inline SVG iconography |
| App promotion | Mobile app call-to-action with device mockups and store badges |
| Footer | Newsletter signup, social links, quick navigation, and contact details |
| Responsive design | Fully adapted layouts across desktop, tablet, and mobile breakpoints |

## Getting started

### Prerequisites

- Node.js 18 or later
- npm

### Installation

```bash
git clone https://github.com/<your-username>/autoride.git
cd autoride
npm install
```

### Run locally

```bash
npm start
```

The app will be available at `http://localhost:3000`.
